Puerto Rico Infographic

What is This Project

This project goal was to create an infographic thatwould communicate a message that is meant to move people towards an action while using only one sources of information. The reason we were asked to use one source to keep the information streamlined. With these limitations in mind I chose a fundraising ad to help raise funds for Puerto Rico’s recovery after Hurricane Maria. For this task I choose to make a social media ad that would be targeted at attracting people who were not entirely familiar with the damage that was done by Hurricane Maria.
